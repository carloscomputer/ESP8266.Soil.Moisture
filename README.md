# ESP8266.Soil.Moisture
My plan to build a secure method to measure the moisture of the soil in plant pots went to a final state, I can release the code from now on.
Hardware:
ESP8266 and Moisture Sensor v2.0, cables

 - The values of dry air and pure water are editable over the WEB-UI you should calibrate the sensor.
 - wifi values and influxdb values you can change after the first setup over the WEB-UI
 - wifi 802.11n standard
 - Germany Country Code (DE)
 - Randomized MAC
